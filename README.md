# android-map-refactoring

## 프로젝트 설명
이 프로젝트는 카카오테크캠퍼스 2기 STEP2 5주차 미니과제로, 의존성 주입과 MVVM 아키텍처 패턴을 도입하여 리팩토링하는 과정을 포함합니다. 기존의 코드를 개선하여 유지보수성을 높이고, 확장 가능한 구조로 변경하는 것을 목표로 합니다.

## 구현 기능 목록
### 1단계 - 의존성 주입
- 데이터베이스를 Room으로 변경
- Hilt를 사용하여 의존성 주입 적용

### 2단계 - 아키텍처 패턴
- MVVM 아키텍처 패턴 적용
  - DataBinding 사용
  - LiveData 사용
- 비동기 처리를 Coroutine으로 변경
